Setup Checklist

Step 1 — Clone the repository

git clone https://github.com/[repo-url]/COMPLIANCEGPT.git
cd COMPLIANCEGPT
pip install -r requirements.txt

Step 2 — Install and test Ollama

# After installing Ollama from ollama.com:
ollama pull gemma3:4b
ollama serve   # leave this running in a terminal

Step 3 — Run your first experiment

python3 app/batch_runner.py \
  --input data/goldcoin_hhs_merged.csv \
  --question-col query \
  --strategy formal \
  --model ollama/gemma3:4b \
  --output /tmp/my_first_run.csv \
  --limit 3

If you see 3 rows with PERMITTED/DENIED verdicts — you're set up correctly.

Step 4 — Inspect the results

import pandas as pd
df = pd.read_csv("/tmp/my_first_run.csv")
print(df[["question", "verdict", "ground_truth"]])

System Requirements

  • Any laptop with 8 GB RAM is sufficient for Gemma3:4b locally.
  • Large models (Llama 70B, Qwen 72B) run on the AI3 cluster or vast.ai GPU servers — not your laptop.
  • Mac, Windows, and Linux are all supported. The codebase is cross-platform.

Python Resources

You don't need to be a Python expert. You need to be functional: load a CSV, loop over rows, call a function, save results. Here are the best free resources to get there fast.

If you're new to Python

Python Crash Course — Eric Matthes

Chapters 1–6 cover everything you need. Available at most libraries or as a PDF online. Focus: variables, lists, dicts, functions, files. Skip the game development chapters for now.

CS50P — Harvard (free on edX)

Python course from Harvard. Excellent production quality. Each problem set takes 2–3 hours. Do Weeks 0–4 if you have time before June 22.

If you know Python basics but want to level up

Automate the Boring Stuff with Python — Al Sweigart

Free at automatetheboringstuff.com. Chapters on CSV, file I/O, and APIs are directly useful for this project.

Corey Schafer — YouTube

Best Python tutorial series on YouTube. Clear, fast, practical. Particularly: the pandas tutorial series (10 videos) is exactly what you need for data analysis.

The 5 things you actually need for this project

# 1. Load a CSV
import pandas as pd
df = pd.read_csv("results.csv")

# 2. Filter rows
wrong = df[df["verdict"] != df["ground_truth"]]

# 3. Write a function
def check_accuracy(df):
    correct = (df["verdict"] == df["ground_truth"]).sum()
    return correct / len(df)

# 4. Loop + print
for i, row in wrong.iterrows():
    print(f"Row {i}: {row['verdict']} (expected {row['ground_truth']})")

# 5. Save results
df.to_csv("analysis_output.csv", index=False)
